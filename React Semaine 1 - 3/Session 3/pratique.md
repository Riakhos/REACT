# Session 3

L'application specs : 

Incrémenter un compteur dans la card produit avec un bouton 'like' 

Bouton 'add to card' permettra de mettre l'article au panier  et d'incrémenter un compteur dans la navbar.

Deux produits de même nom ne peuvent pas être mis au panier. 

Clicker sur le bouton 'card' permettra d'afficher une sidebar/dropdown référençant les produits mis au panier. 

Une icone trash permettra de retirer un élément du panier et désélectionné le produit dans la page principale. 

Chaque produit sélectionné aura un bouton vert et rouges dans le cas contraire (variante bouton disabled et enabled)
