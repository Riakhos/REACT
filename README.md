# REACT

Ce dépôt regroupe deux applications React distinctes :

- **allo-cine** : Application de gestion et de découverte de films.
- **shop-online** : Application de boutique en ligne avec panier et gestion des articles.

## Structure du dépôt

- Branche `allo-cine` : code source de l’application Allo-Ciné.
- Branche `shop-online` : code source de l’application Shop-Online.

## Démarrage rapide

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/REACT.git
   ```
2. Basculez sur la branche souhaitée :
   ```bash
   git checkout allo-cine
   # ou
   git checkout shop-online
   ```
3. Installez les dépendances :
   ```bash
   npm install
   ```
4. Lancez le projet :
   ```bash
   npm run dev
   ```
